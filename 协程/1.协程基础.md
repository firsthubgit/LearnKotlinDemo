### 1
协程是轻量级的线程。它在某些CoroutineScope上下文中与launch协程构建器一起启动。

### 2
runBlocking协程构建器，会阻塞主线程，直到runBlocking内部的协程执行完毕。
```kotlin
fun main(){
    println("Hello") //主线程中代码会立即执行
    runBlocking{ //阻塞了主线程
        delay(1000)
    }
}
```
或
```kotlin
    // runBlocking作为用来启动顶层主协程的适配器
    fun main() = runBlocking<Unit>{

    }
```
